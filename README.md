  

##Requirements 



#####this app requires :

       gtk+-3.0
       gtksourceview-3.0
       vte-2.90

#####install them before start compiling



  
##install
  
  to complie and install this app use setup script
  
			  ./setup.sh <command>
			  
			  
  
##Uninstall  
  
  
  to remove application just execute this command
  
			  csimple_uninstall
